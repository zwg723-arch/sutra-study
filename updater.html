🚀 一鍵推送到 GitHub

在 Claude 修改完畢後，用這個頁面直接更新網站
Vercel 會自動重新部署（約 1 分鐘）

📱 使用步驟
  1. 在 Claude 對話中，點擊最新的 index.html 檔案 → 下載
  2. 回到這個頁面，點「📋 從剪貼簿貼上」或直接貼到下方文字框
  3. 輸入你設定的推送密碼
  4. 按「🚀 推送到 GitHub」— 完成！

⚙️ Vercel 環境變數設定（一次性）
需在 Vercel → Settings → Environment Variables 加入：
GITHUB_TOKEN — GitHub Personal Access Token（repo 權限）
GITHUB_OWNER — 你的 GitHub 用戶名
GITHUB_REPO — Repository 名稱（如 sutra-study）
PUSH_SECRET — 自訂一個推送密碼（隨機字串）
